jQuery AJAX Functions


This bookmark focuses on use jQuery AJAX functions to make calls to the Google Books API. Users can use the search box to input terms and see a minimally formatted list of results. Clicking a result will bring up the page details below the search results. The search results are paginated for more compact fit on the page. My personal bookshelf can be accessed at any time by clicking the 'My Bookshelf' button.
