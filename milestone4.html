Milestone 4


User Interface Refinement


This milestone focuses on improving the user interface. This page uses Mustache.js to create html templates that are dynamically populated with variables, making it easy to separate the view and model logic. Users can choose whether to view search results in a list or grid format. This page also uses a "load more" implementation as an alternative to pagination. The user can also choose to see a bookshelf of "Editor's recommendations" at any time, and switch back to their search results without losing their place.
