This site is meant to be a living portfolio to exhibit my course work and things I've learned in IT4203 Advanced Web Development, taught by Richard Windland. Throughout the semester this project will have four milestone deliverables. Each one will focus on a different aspect of web technologies that will culminate in book search web app. This web app will be connected to the Google Books API for search results. By the end of the semester, any user should be able to browse books based on key attributes such as author, subject, or publisher. Users will be able to see complete book information for any of the results selected, and even create and save a list of books for later use.
